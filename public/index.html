Chuck Norris Jokes MCP Server

A Model Context Protocol server for the Chuck Norris Jokes API

About

This server implements the Model Context Protocol (MCP) to provide AI assistants with access to the Chuck Norris Jokes API.

With this MCP server, AI assistants like Claude can retrieve random Chuck Norris jokes, get jokes from specific categories, list all available categories, and search for jokes by text.

Available Tools

get-random-joke

Get a random Chuck Norris joke

{ }

get-joke-from-category

Get a random Chuck Norris joke from a specific category

{ category: string }

get-categories

Get all available Chuck Norris joke categories

{ }

search-jokes

Search for Chuck Norris jokes containing specific text

{ query: string }

Using with Claude

To use this MCP server with Claude, you'll need to configure it in the Claude Desktop app. See the MCP Quickstart for Claude Desktop Users for detailed instructions.

For Developers

The MCP server is deployed on Vercel and uses the MCP TypeScript SDK. The source code is available on GitHub.
